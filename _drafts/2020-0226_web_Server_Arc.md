웹브라우저 동작 순서 

1. 웹사이트 접속 URL 입력
2. 각 도메인 IP 변환작업
	로컬 DNS 캐시 확인
	hosts 파일 참조
	DNS 서버 질의 -> 해당 도메인의 아이피주소를 알아낼떄까지 DNS서버 서로 질의 한다.
3. HTTP 요청 메세지 제작

TCP/IP 통신을 이용하여 웹서버에 접속
	TCP 연결의 특징이 적용됨
	3 Way hand shake 


클라이언트  --- TCP/IP ---- 웹서버 ---- 데이터베이스
